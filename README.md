Simiquan
========

给android多一点的可能，android界面过场动画效果新尝试。


自定义界面转场动画实现思路（Activity跳转）

通过将目标Activity主题设置为透明，在跳转完成之后开始实现转场动画。

采用android属性动画框架，实现控件的移动和拓展动画效果。
